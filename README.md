# 📊 Customer Churn Analytics Dashboard (Power BI)

A Power BI dashboard designed to identify churn drivers and support targeted retention strategies in a telecom environment.


---

## 🚀 Business Problem

Subscription-based telecom providers face significant revenue loss from customer churn. This analysis was built to:

- quantify churn rate and segment risk
- identify behavioral and demographic churn drivers
- highlight high-impact retention opportunities

---

## 📌 Key Insights

- **Churn rate:** 26.54%
- **Highest-risk segment** Customers aged 50+
- **Contract risk:** Month-to-month contracts account for over 40% of churn
- **Payment behavior:** Electronic check users show higher churn vs. auto-pay users
- **Product usage:** Customers without bundled services churn at higher rates

## 🧠 Analytical Approach
- Cleaned and transformed raw data using Power Query
- Created calculated measures and KPIs using DAX
- Engineered features including:
  - Age groups
  - Customer status (active vs churned)
  - Tenure
  - Derived churn rate metrics
- Structured data model to support flexible segmentation and drill-down analysis

---

## 📁 Dataset

- **Rows:** 56,344
- **Core fields:** gender, age, contract, monthly charges, services, churn
- **Derived fields:** age group, tenure, location, churn rate

---

## 🎯 Business Impact

This dashboard enables:

- identification of high-risk customer segments for targeted retention campaigns
- evaluation of contract and payment structures influencing churn
- prioritization of bundling strategies to improve customer stickiness
